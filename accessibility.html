Accessibility Statement

Pointless Polygon Studio is committed to making this website accessible in accordance with the Web Content Accessibility Guidelines (WCAG) 2.2, Level AA.

Conformance Status

This website is designed to be largely compliant with WCAG 2.2 Level AA. Accessibility was considered during design and implementation, with a focus on semantic HTML, readability, and keyboard usability.

Accessibility Features

  • Semantic HTML structure using headings, landmarks, and lists
  • Text alternatives (alt attributes) for all meaningful images
  • Sufficient color contrast between text and background
  • Keyboard-accessible navigation and controls
  • Responsive layout that supports zooming and different screen sizes
  • No autoplaying audio or video

Known Limitations

Despite best efforts, some limitations may exist:

  • Embedded third-party content (e.g. YouTube links) may not fully meet accessibility requirements, as they are controlled by external providers.
  • The website has not undergone formal external accessibility certification or a full VPAT audit.

Accessibility was evaluated using:

  • Manual testing with keyboard navigation
  • WAVE - the automated tool
  • Visual inspection for contrast and readability

Feedback and Contact

If you encounter any accessibility barriers or have suggestions for improvement, please contact us via the contact page.
